### 3. 前 3 秒钩子
- **钩子类型**：**视觉猎奇钩 + 痛点提问钩**（视觉上是极度解压的“抽拉动作”，听觉上是“为什么别人手机比你高级”的灵魂拷问）。
- **为什么能留住用户**：人类本能喜欢看规整、丝滑的机械/手工动作（类似解压视频）；同时文案击中了虚荣心和审美痛点。
- **可优化的 3 个开头版本（针对不同人群测试）**：
  1. **极反差测试钩**：“千万别眨眼！”（直接放 43 秒锤子砸屏幕的画面，然后倒放回到贴膜前），适合追求质量的受众。
  2. **痛点放大钩**：“你的手机膜是不是又贴废了？”（特写满是气泡和灰尘的屏幕，直接对比3秒贴好的新膜），适合手残党。
  3. **场景代入钩**：“坐地铁老被偷看手机？”（直接展示 48 秒的防窥侧面黑屏效果），适合对隐私敏感的通勤族。

- **商品出现次数和出现方式**：全程出镜，且永远位于画面的视觉中心。从包装舱 -> 贴膜过程 -> 成品测试 -> 包装盒，全链路展示。

  - 无尘自动贴膜（一拉一按解决贴膜难题）。
  - 疏油层防指纹、防水（屏幕顺滑）。
  - 听筒防尘防渗水。
  - 28度防窥（保护隐私）。
  - 高铝玻璃防爆抗摔。

### 5. 脚本结构
- **脚本模型**：采用经典的 **Hook（吸引）— Problem（痛点）— Solution（方案）— Proof（自证）— CTA（行动呼唤）** 模型。
- **口播话术逻辑**：开局制造焦虑（你还在用过时的膜） -> 给出降维打击方案（大家都在用自动贴膜舱） -> 夸张的易用性比喻（曾祖父都会贴） -> 密集输出卖点并配合测试 -> 限时优惠逼单。
- **CTA**：视频结尾展示完整品牌包装，配以“不要错过新优惠”，明确引导转化。

### 6. 视觉语言
- **构图**：标准的短视频竖屏中心构图，画面主体（手机）几乎填满中下部三分之二的黄金视觉区。
- **景别**：大量使用**极近特写（Macro shots）**，如水滴滑落、听筒防尘、边缘防窥等，这种景别在电商视频中极具说服力，能放大产品细节。
- **运镜**：以固定机位的第一人称主观视角（POV）为主，增强代入感，让观众感觉是自己在操作。
- **剪辑节奏**：快节奏，平均 3-4 秒切换一个画面或测试，绝不拖泥带水，全程高能刺激多巴胺。
