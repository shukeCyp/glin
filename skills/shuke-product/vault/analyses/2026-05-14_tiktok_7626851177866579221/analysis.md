### 3. 前 3 秒钩子
- **钩子类型**：**“开箱动作” + “价格悬念/反差”**。视觉上是原生态的拆快递，听觉上是“吐槽”价格奇怪（暗示极具性价比）。
- **为什么能留住用户**：
  1. 拆快递是全人类共通的爽点，动作本身具有动态吸引力。
  2. 话术切中消费者“捡漏”心理，“没多想就买了结果很惊喜”是经典的种草带货句式，引发用户想看看到底是什么好东西。
- **为什么不能留住用户**：如果受众对“草莓图案”这种过于甜美的元素完全不感冒，可能会在商品露出的第一秒划走。
- **可优化的 3 个开头版本**：
  1. **容量痛点前置**：前3秒快速把一堆书本/电脑/化妆包塞进包里（“到底多能装？打工人/学生党的天选通勤包！”）。
  2. **颜值穿搭前置**：直接放出0:10秒对着镜子绝美的上身搭配画面，然后再倒叙拆箱（“被问疯了的草莓包，居然只要一杯奶茶钱？”）。
  3. **暴力测试/质感前置**：前3秒给面料、走线、或者外置口袋塞入手机的极度特写，强调材质（“几十块钱买到几百块的质感，我不允许你们不知道！”）。

### 4. 商品与卖点
- **商品出现次数和方式**：贯穿全片 100% 出现。依次为：带包装出箱 -> 拆包展示 -> 窗台静态展示 -> 撑开看内里 -> 镜前装物展示 -> 镜前上身展示。轨迹极其顺畅。
- **明确卖点（口播传达）**：价格便宜（性价比极高）、质量远超预期、限时特价。
- **隐含卖点（视觉传达）**：
  - **大容量且立体**：放在窗台上不塌陷，内部空间深。
  - **收纳合理**：前置两个大口袋，完美适配手机和卡包。
  - **颜值百搭**：草莓印花清新，肩带长度适中，搭配浅色系衣服很出片。
- **可信度证据**：
  1. 剪开塑料包装的真实感。
  2. 窗台自然光下的内衬特写（无滤镜掩饰）。
  3. 真实的装物动作（手机放入口袋严丝合缝）。
  4. 居家生活环境（非刻意搭建的影棚），降低防备心理。

### 5. 脚本结构
- **脚本模型**：**SCQA变体模型 (情境-冲突-疑问-解答) + 强AIDA促单**
  - **A (Attention)**：拆快递 + “谁定的这个价格”引起注意。
  - **I (Interest)**：展示窗台自然光下的质感和超大内部空间。
  - **D (Desire)**：真人演示外口袋装手机，上身搭配好看，激发占有欲。
  - **A (Action)**：直接点明“去购物车”、“特价”、“马上断货”，促发冲动消费。
- **口播话术特征**：采用“闺蜜分享式”语气。使用“我发誓(Se los juro)”、“说真的(en serio)”、“毫不夸张(Literal)”等极具主观情绪感染力的词汇。
- **CTA (行动号召)**：明确指向“小黄车/购物车 (al carrito)”，并叠加了稀缺性和紧迫感（“正在特价”、“不会持续太久”）。

### 6. 视觉语言
- **构图**：以中心构图为主，商品始终处于视觉画面的绝对C位（正中央）。
- **景别**：中景（交代开箱环境/上身效果）与特写（交代包包内里/装物细节）交替使用，逻辑清晰。
- **运镜**：第一视角手持运镜与三脚架固定镜头结合。展示内部时手持推进，代入感强；上身展示时固定镜头，让观众视线聚焦在搭配上。
- **剪辑节奏**：快节奏，每2-3秒切换一个场景（纸箱->窗台->镜前装物->镜前背挎），没有冗余帧，非常符合短视频完播率要求。
