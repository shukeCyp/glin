### 4. 商品与卖点
- **商品出现次数和出现方式**：3次。带包装展示（背景中） -> 唇线笔脱壳试色 -> 口红脱壳试色。
- **明确卖点**：黄油般的顺滑质地（滑顺不卡纹）、极高显色度。
- **隐含卖点**：同色系套装搭配（解决用户自己配色的烦恼）、深红色系显白、小众平价宝藏。
- **可信度证据**：**一镜到底的手背试色**。没有任何剪辑跳跃的涂抹过程是彩妆最好的视觉证据，膏体边缘的平滑度直接证明了没有结块或干涩。

- **脚本模型**：**UGC发现 -> 沉浸式试色证明 -> 情绪化安利** (Discovery -> Proof -> Emotion)。

### 6. 视觉语言
- **构图**：全程居中构图，焦点明确，背景做虚化处理。
- **景别**：中景（交代环境）快速过渡到特写（Product shot）和微距（Macro shot）。
- **运镜**：第一人称手持运镜，带有轻微的呼吸感抖动，增加了视频的“非商业化”真实属性。
- **剪辑节奏**：快节奏，全长仅10秒，没有废镜头，动作衔接紧凑（拿包装-画线-涂色-展示）。

### 8. 可复刻方案（适用于国内美妆带货）
- **脚本大纲**：
    1. (0-2s) 从拥挤的化妆包里翻出产品（吸睛）。
    2. (2-5s) 手背快速画唇线+涂口红（视觉证明）。
    3. (5-8s) 镜头拉近展示膏体滋润度，手指抹开（强化质感）。
    4. (8-10s) 指向屏幕下方引导下单（转化）。
- **镜头清单**：
    - 镜头1：俯拍，手在杂乱化妆品中拿起目标口红套装。
    - 镜头2：手背特写，唇线笔画一笔（需保证无卡顿）。
    - 镜头3：手背特写，口红在唇线旁涂抹（重点打光，体现膏体光泽）。
    - 镜头4：手指晕染边缘，贴上卖点字幕+购物车指示箭头。
