### 5. 脚本结构
*   **脚本模型：** PAS框架 (Problem 痛点 -> Agitate 搅动/恶化 -> Solution 解决) + 强视觉Demo。
*   **口播话术逻辑：**
    *   **引出痛点：** 衣服粘毛、发臭，因为你在用脏水洗衣服。
    *   **揭露真相：** 机器深层藏污纳垢（配合拆机画面）。
    *   **抛出产品：** 专门针对此痛点的清洁片，便宜量大。
    *   **使用方法：** 丢一粒，空转。
    *   **见证奇迹：** 排出黑水，内筒如新。
    *   **CTA（呼吁）：** 点击小黄车，第一次用就有效。
*   **CTA：** 强指令性，“Te dejo el enlace en el carrito para que las consigas” (我把链接留在购物车里供你获取)，并附加效果承诺兜底。

### 6. 视觉语言
*   **构图与景别：** 大量使用**特写（Macro/Close-up）**。特写黑水、特写脏衣服、特写陈年老垢、特写排出的泥水。目的是无限放大“脏”的视觉冲击力。
*   **运镜：** 手持跟拍风格，带有轻微晃动，营造出非精美广告的“原生态真实感”（UGC属性），拉近与用户的距离。
*   **剪辑节奏：** 快节奏跳剪。前15秒高频切换脏水、脏衣、脏内筒，不断刺激视觉神经；后半段产品展示和排污画面停留时间较长，让情绪释放（解压）。

### 8. 可复刻方案
*   **脚本大纲：**
    1.  **痛点/共鸣 (0-3s)：** 拿出一件刚洗完但发酸的衣服，或者粘满毛屑的衣服。
    2.  **揭秘/恐吓 (3-10s)：** 镜头对准看似干净的洗衣机内筒，然后翻开胶圈或者展示修理工拆解的超脏内筒。
    3.  **产品出场 (10-15s)：** 亮出泡腾片，“不用请师傅拆洗，只要这小小一粒”。
    4.  **演示过程 (15-25s)：** 丢入洗衣机，注水，泡腾片剧烈沸腾冒泡（体现活性成分）。
    5.  **高光验证 (25-30s)：** 拍摄排污管排出黑色浊水的特写，或者打捞出水面的黑毛垢。
    6.  **转化呼吁 (30s+)：** 展示光亮的内筒，强调一盒能用很久，引导左下角下单。
*   **镜头清单：** 脏衣服特写、嫌弃表情、翻开洗衣机胶圈黑垢特写、泡腾片入水沸腾特写、排污水管特写、干净内筒全景。
