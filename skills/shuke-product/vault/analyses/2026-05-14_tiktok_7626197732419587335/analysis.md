### 3. 前 3 秒钩子分析
- **钩子类型**：**动作猎奇 + 利益直给型钩子**。
- **为什么能留住用户**：常规衣服没有连体面罩，模特行云流水地拉起面罩戴上帽子，动作本身具有新奇感（类似于忍者或特种兵），能引发好奇心。配合醒目的“PACK DE 3 PLAYERAS”字幕，直接击中价格敏感型用户的爽点。
- **可优化的 3 个开头版本 (A/B Test 建议)**：
  1. **痛点放大版**：前1秒展现满头大汗、脖子被晒红的痛苦表情，第2秒迅速切到穿上这款衣服拉上面罩的从容感。（制造对比）
  2. **算账/价格钩子版**：直接在画面打出极其低廉的单件折算价格，口播：“不到一杯奶茶钱，拿下3件防晒神衣！”（强化利益点）
  3. **暴力测试版**：开篇直接用水蒸气透过衣服面料（证明透气），或者用紫外线感应卡放在衣服下测试（证明防晒），再引出商品。（强化功能性）

### 4. 商品与卖点
- **商品出现次数和方式**：贯穿全片（100%覆盖）。通过“上身试穿（灰/蓝）”、“手持展示全貌（灰/黑）”、“局部特写（面料）”三种维度全方位展示。
- **明确卖点（口播+文字）**：3件装（性价比）、柔软、透气、一体化面罩、带帽、长袖（防晒覆盖率高）。
- **隐含卖点（视觉传递）**：轻薄（随风飘动）、弹力好（拉扯不变形）、多色可选（灰黑蓝）、户外百搭。

### 5. 脚本结构
- **脚本模型**：**FABE 简化版结构** (Feature特性 - Advantage优势 - Benefit利益 - Evidence证据) + 强转化尾部。
  - *特性*：连帽连面罩的长袖。
  - *利益*：3件装超低价。
  - *证据*：拉扯透气网眼面料。
  - *转化*：指引点击购物车。
- **口播话术**：典型的“叫卖式”带货话术。语速快，情绪饱满，高频使用诸如“super precio（超低价）”、“muy buena calidad（质量很好）”等促销词汇。
- **CTA (Call to Action)**：视觉（向下指的手势、红色箭头、动图贴纸）与听觉（口播“点击橙色购物车”）高度统一，转化指令极其清晰。

### 6. 视觉语言
- **构图**：以人物为中心的居中构图，商品始终处于视觉C位。
- **景别**：中景为主（展示上身效果）+ 微距特写（打透面料细节），景别搭配合理，无废镜头。
- **运镜**：采用手持微晃的跟拍镜头。这种**UGC（用户生成内容）原生感**非常适合短视频流，降低了广告的生硬感，增加了真实评测的信任度。
- **剪辑节奏**：快节奏，每3-4秒切换一个场景/颜色，信息密度高，防止用户视觉疲劳滑走。贴纸的加入增加了趣味性和网感。
