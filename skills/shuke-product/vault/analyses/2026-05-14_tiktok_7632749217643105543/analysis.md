### 3. 前 3 秒钩子
- **钩子类型**：场景带入型 + 粉丝粘性钩子（GRWM式开头）。
- **为什么能/不能留住用户**：
  - *能留住*：博主个人IP属性强，车内场景具有一定的私密感和真实感，像是在和闺蜜视频聊天。
  - *不能留住*：前3秒**没有出现产品**，也没有抛出强烈的痛点或利益点。对于非粉丝的泛流量来说，前3秒略显平淡，容易划走。
- **可优化的 3 个开头版本（针对泛流量提升完播）**：
  - *视觉冲击前置*：第一秒直接放戴好假发的惊艳状态，第二秒切回干瘪短马尾。“从干瘪发量到女团高颅顶，我只用了3秒钟。”
  - *痛点提问开头*：手里拿着假发晃动。“还在因为发量少不敢扎马尾吗？这个TikTok爆款假发救了我！”
  - *动作悬念开头*：视频一开始直接是“撑开爪夹、往头上夹”的动作特写，边夹边说：“我倒要看看这个网传神器到底多好用。”

### 4. 商品与卖点
- **商品出现次数和方式**：3次核心展示（手持展示总体积 -> 梳理展示发质 -> 实际佩戴展示效果）。
- **明确卖点**：
  - 波浪卷发，造型好看（通过梳理和上头展示）。
  - 佩戴方便（通过一镜到底的佩戴动作展示）。
- **隐含卖点**：
  - 牢固不掉（通过佩戴后大幅度摇头验证）。
  - 融入度高（即使有轻微色差，整体视觉依然协调）。
- **视觉证明点（Proof Points）**：
  - **梳子梳理流畅** = 发质好，不毛躁。
  - **原发与假发对比** = 解决发量痛点的强力对比。
  - **摇头测试** = 爪夹咬合力强，适合日常或工作活动。

### 5. 脚本结构
- **脚本模型**：**情境（Context） - 问题（Problem） - 方案（Solution） - 验证（Proof） - 行动（CTA）**
- **口播话术亮点**：话术非常口语化、生活化（如“我原本只有3块钱的头发”形容发量少，非常生动）。敢于暴露瑕疵（白发、色差），这种“反向营销”反而极大地拉升了用户的信任感。
- **CTA (Call to Action)**：明确指出“橘色购物车（carrito naranja）”的位置，并配合手指动作，引导路径极度清晰。

### 6. 视觉语言
- **构图**：标准的竖屏带货构图，人物居中，半身出镜，胸部以上和头部留有足够空间展示发型变化。
- **景别**：中近景（展示整体穿搭和发型）与面部特写（展示白发闲聊时）自然切换（通过博主自身身体前后倾斜实现，而非剪辑）。
- **运镜**：固定机位，稳定性好，不晕眩，适合教程和展示类内容。
- **剪辑节奏**：极少剪辑点，几乎是一镜到底。这种“不修饰”的粗糙感正是原生短视频带货的优势，极大地增强了“眼见为实”的说服力。
