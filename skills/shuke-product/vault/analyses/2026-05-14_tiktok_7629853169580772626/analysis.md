- **脚本模型**：**P-S-R-A 模型** (Problem 痛点/场景 -> Solution 方案 -> Resolve 消除顾虑 -> Action 促单)。

- **构图/景别**：90%以上是极近景（面部大特写）。在竖屏短视频中，这种极近景能放大五官的变化，让用户看清“毛流感”，非常适合美妆个护类投放。

- **剪辑节奏**：快节奏跳剪（Jump-cut）。去除了所有等待、犹豫的无效帧，动作连贯（洗脸-调色-涂抹-擦除一气呵成），完播率有保障。
