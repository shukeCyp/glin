### 3. 前 3 秒钩子
- **钩子类型**：**“反差/猎奇音频” + “解压视觉”双重钩子**。视觉上是极其顺滑的刮胡子解压过程，听觉上却是关于“剃私密部位被割伤”的痛点描述。
- **为什么能留住用户**：人类对“疼痛（割伤）”和“私密话题”有天然的敏感度（音频）；同时，看着浓密的胡须被一扫而光会产生心理上的爽感（视觉）。这种音画错位在短视频平台极易引发评论和完播。
- **可优化的 3 个开头版本**：
  1. **极限测试钩子**：用该剃须刀在充气的气球表面刮剃，气球不破，直接证明“不割肉”的绝对安全承诺。
  2. **痛点对比钩子**：分屏展示——左边是传统手动剃须刀流血贴创可贴的下巴，右边是使用该产品极其丝滑无红肿的下巴。
  3. **场景痛点钩子**：男子在车里马上要见客户，掏出这个迷你剃须刀，3秒钟干干净净，强调“便携急救”属性。

### 4. 商品与卖点
- **商品出现次数和出现方式**：商品贯穿全片100%的时间。通过真人试用（脸部）、手持展示、桌面静物、开盖内部展示等全方位多角度露出。
- **明确卖点**：不割肉/不夹须（口播提及）、易清理（刷子扫除胡茬画面）、便携小巧（单手握持）、Type-C充电（底部接口展示）。
- **隐含卖点**：高颜值（金属质感的银色与磨砂黑色）、全身水洗/防水（口播提及淋浴可用，虽未直观展示但通过刀头材质暗示）。
- **可信度证据**：拆开刀头看到里面的双环刀片；用刷子扫出真实的碎胡茬；不同年龄段（大胡子男和小鲜肉）的使用画面证明不挑人群。

### 5. 脚本结构
- **脚本模型**：**PAS模型 (Problem 痛点 - Agitation 刺激 - Solution 方案)** 变形版。
  - *痛点*：怕割伤、怕夹肉（前3秒）。
  - *方案*：出示迷你剃须刀及刀网特写。
  - *证明*：开箱、清理、充电口展示。
  - *促单*：红色箭头+限时折扣。
- **口播话术**：典型的“问题引入 -> 卖点罗列 -> 痛点解决 -> 强力逼单”的短平快爆款文案。
- **CTA (行动号召)**：非常野蛮且有效。视觉上不仅有“Subscribe”弹窗，更关键的是使用了贯穿视频后半段的**巨大红色下指箭头**，配合口播“点击橙色购物车（TikTok特有词汇）”，目标极其明确。

### 6. 视觉语言
- **构图与景别**：大量使用**微距特写 (Macro)** 和 **怼脸特写 (ECU)**。这在电商视频中是为了占据用户整个手机屏幕，强迫用户将注意力集中在产品细节和使用效果上，摒弃一切背景杂乱。
- **运镜与剪辑节奏**：全程手持拍摄，带有轻微呼吸感，增加了UGC（用户生成内容）的真实感。剪辑节奏极快，平均 **1.5秒切换一个镜头**。这种“信息轰炸”式的剪辑能有效降低用户的滑动流失率。
